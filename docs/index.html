Step 1: Initialization and Setup

This project is a step-by-step tutorial for building and publishing a multilingual website using Go (Golang), hosted for free on GitHub Pages.

  1. Git, Go and VS Code installed
  2. GitHub account created (michelbr84)
  3. Project initialized and pushed to GitHub
  4. Basic static version created and published

Step 2: Multilingual Support

We implemented dynamic i18n in Go, then rendered static pages for each language to support GitHub Pages publishing.

  • Translations defined in a Go map
  • One HTML template used with dynamic placeholders
  • Static files generated for English, Portuguese, and Spanish
  • Language switcher implemented with static links

This is the English version of the site. You can switch languages using the buttons above.
