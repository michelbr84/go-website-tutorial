Etapa 1: Inicialização e Configuração

Este projeto é um tutorial passo a passo para criar e publicar um site multilíngue com Go (Golang), hospedado gratuitamente no GitHub Pages.

  1. Instalado Git, Go e VS Code
  2. Conta do GitHub criada (michelbr84)
  3. Projeto inicializado e enviado para o GitHub
  4. Versão estática básica criada e publicada

Etapa 2: Suporte a Múltiplos Idiomas

Implementamos i18n dinâmico em Go e depois geramos páginas estáticas para cada idioma, garantindo compatibilidade com GitHub Pages.

  • Traduções definidas em mapa Go
  • Template único com variáveis dinâmicas
  • Arquivos HTML estáticos gerados para inglês, português e espanhol
  • Botões de troca de idioma entre páginas

Esta é a versão em português do site. Você pode trocar de idioma usando os botões acima.
