Paso 1: Inicialización y Configuración

Este proyecto es un tutorial paso a paso para crear y publicar un sitio web multilingüe con Go (Golang), hospedado gratuitamente en GitHub Pages.

  1. Instalación de Git, Go y VS Code
  2. Cuenta de GitHub creada (michelbr84)
  3. Proyecto inicializado y enviado al GitHub
  4. Versión básica y estática creada y publicada

Paso 2: Soporte Multilingüe

Se implementó i18n dinámico en Go y luego se generaron archivos estáticos para cada idioma compatible con GitHub Pages.

  • Traducciones definidas en mapas de Go
  • Una sola plantilla HTML con contenido dinámico
  • Archivos HTML estáticos para inglés, portugués y español
  • Botones para cambiar entre idiomas

Esta es la versión en español del sitio. Puedes cambiar el idioma con los botones arriba.
