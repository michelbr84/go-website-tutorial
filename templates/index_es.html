Un tutorial paso a paso para crear y publicar un sitio web multilingüe con Go (Golang) usando GitHub Pages gratuitamente.

Este sitio es open source. Mejora esta página.

Objetivo del Proyecto

El propósito es construir un sitio web con Go como un tutorial en 10 etapas, publicado gratuitamente en GitHub usando este mismo proyecto como ejemplo.

  1. Tutorial en 10 pasos sobre cómo crear y publicar un sitio web con Go.
  2. Público objetivo: personas que hablan inglés, español o portugués.
  3. Asistencia para definir o aplicar identidad visual.
  4. Cada paso publicado y archivado por separado.
  5. Se podrá ver el resultado de cada etapa en páginas específicas.
  6. 10 lanzamientos progresivos hasta el sitio completo.
  7. Después del Paso 10, no habrá más actualizaciones inicialmente.
  8. Sitio optimizado para motores de búsqueda (SEO).
  9. No se necesita integración con otros sistemas.
  10. El proyecto será totalmente gratuito, salvo el tiempo invertido.

Esta primera etapa será archivada como definitiva. No se sobrescribirá. Las próximas etapas se agregarán como archivos y carpetas nuevas.

Documentación Inicial

  • Cuenta GitHub: creada como michelbr84.
  • Instalaciones: Git, Go (1.24.1), VS Code y extensiones (Go, GitHub, Live Server).
  • Configuración: módulo inicializado con go mod init en go-website-tutorial.
  • Framework: Gin instalado con go get -u github.com/gin-gonic/gin.
  • Servidor local: ejecutando con go run main.go en http://localhost:8080.

Resumen del Paso 1: Inicialización y Configuración Básica

  1. Entorno: Git, Go y VS Code configurados.
  2. Repositorio: creado y sincronizado con GitHub (link).
  3. Código: archivo main.go con ruta básica y servidor funcional.
  4. Servidor: probado exitosamente en entorno local.
  5. Publicación: versión inicial publicada con GitHub Pages.
