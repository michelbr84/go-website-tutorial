Um tutorial passo a passo para criar e publicar um site multilíngue com Go (Golang) gratuitamente usando o GitHub Pages.

Este site é open source. Contribua ou melhore esta página.

Objetivo do Projeto

Este projeto visa criar um site com a linguagem Go. O site será um tutorial em 10 etapas mostrando como criar e publicar um site gratuitamente no GitHub, usando este mesmo site como exemplo.

  1. Tutorial em 10 etapas para criar e publicar um site com Go via GitHub.
  2. Público-alvo: interessados em Go, com suporte em inglês, espanhol e português.
  3. Auxílio na criação ou aplicação de identidade visual.
  4. Cada etapa é publicada individualmente e pode ser visualizada separadamente.
  5. Haverá uma página para acompanhar como ficou cada etapa concluída.
  6. Serão 10 lançamentos até o site final.
  7. Após a Etapa 10, o conteúdo não será mais atualizado inicialmente.
  8. Site otimizado para mecanismos de busca (SEO).
  9. Sem integração com outros sistemas.
  10. Todo o projeto será gratuito, apenas com investimento de tempo.

Cada etapa será arquivada no GitHub como um bloco separado, e futuras etapas não substituirão esta. A Etapa 1 está finalizada.

Documentação Inicial

  • Conta GitHub: Criada com usuário michelbr84.
  • Instalações: Git, Go (1.24.1), VS Code e extensões (Go, GitHub, Live Server).
  • Configuração: VS Code aberto em go-website-tutorial, módulo iniciado com go mod init.
  • Framework: Gin instalado com go get -u github.com/gin-gonic/gin.
  • Servidor de teste: go run main.go rodando em http://localhost:8080.

Resumo da Etapa 1: Inicialização e Configuração Básica

  1. Ambiente: Git, Go e VS Code configurados.
  2. Repositório: Criado e sincronizado com GitHub (link).
  3. Código: Arquivo main.go criado usando Gin.
  4. Servidor: Rodando localmente com rota básica funcionando.
  5. Publicação: Primeira versão hospedada com GitHub Pages.

Essa etapa foi concluída com sucesso. No GitHub, será preservada separadamente na estrutura de pastas. As próximas etapas serão implementadas em diretórios novos.
