Project Objective

This project aims to create a website using the Go programming language. The website serves as a 10-step tutorial to build and publish a website on GitHub for free, using this very site as an example.

  1. Create a 10-step tutorial on building and publishing a website with Go via GitHub, free of charge.
  2. Target users interested in learning to create websites with Go, available in English, Spanish, and Portuguese.
  3. The tutorial also guides users on implementing or creating a visual identity, since the company lacks one.
  4. The site's functionality is to present a step-by-step tutorial where each stage is published and reviewed individually.
  5. Users can view each completed stage on separate pages showing the results of that particular process.
  6. The project will be released in 10 stages, with each stage published until the complete website is finished.
  7. After the 10th stage, the content will initially not be updated.
  8. The site must be optimized for search engines (SEO).
  9. No integration with existing company systems is required.
  10. Everything must be done for free, with the only investment being time.

At the end of each stage, the GitHub repository will be updated and the website published as-is, resulting in 10 semi-complete versions until the final version is released.
